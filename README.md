## 一、整体介绍

![image-20260715162837708](CUITSU.assets/image-20260715162837708.png)

成都信息工程大学校学生会（ **Chengdu University of Information Technology Student Union**，简称 **SUITSU**）是在校党委统一领导、校团委具体指导下成立的全校性最高学生自治组织。组织严格遵循“自我教育、自我管理、自我服务、自我监督”的工作准则，立足全校龙泉、航空港两校区，面向全体在校学生开展工作。

SUITSU始终秉持“对外优质服务，对内优秀成长”的核心工作宗旨，承担思想引领、校园文化建设、学生权益维护、学风建设、师生沟通桥梁搭建等核心任务。作为连接学校职能部门与广大同学的核心纽带，校学生会立足同学需求、聚焦校园实际，常态化收集并反馈学生诉求，策划开展各类校级品牌活动，服务全校学生成长成才，同时协助学校落实学生管理与校园文化建设相关工作，是学校学生工作体系中不可或缺的重要学生组织。

## 二、下设五大部门

### 1. 办公室

学生会核心内务统筹部门，负责日常文书归档、会议筹备、物资经费管理、干部考核及部门协调工作，保障学生会各项工作规范、有序开展。

### 2. 活动部

校园各类校级活动的核心负责部门，主要承担文体、学风、主题教育等校园活动的策划、落地与统筹工作，丰富校园文化生活。

### 3. 生活权益部

专注学生校园生活服务与权益保障，收集整理学生各类生活、学习诉求，对接学校相关部门沟通反馈，开展各类便民及校园文明建设活动。

### 4. 宣传与媒体部

学生会官方对外宣传平台，负责新媒体账号运营、活动宣传物料制作、活动影像记录及内容推送，传递校园正能量与学生会工作动态。

### 5. 组织联络部

核心负责校级活动对外外联赞助、物资资源对接，保障大型活动物资与经费支撑；同时兼顾校际学生组织交流、校内院会联动对接，辅助统筹大型学生会议相关联络工作。
